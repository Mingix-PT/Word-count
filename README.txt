Lưu ý: chương trình bai33.c chạy tuần tự một lần 3 tính năng không lặp lại, còn chương trình
bai33menu.c có chứa menu và sẽ được dùng lại tính năng.

Chương trình bai33menu.c có thêm tính năng khi dùng lại tính năng 1 sẽ xóa dữ liệu từ file cũ
và thay bằng dữ liệu của file hiện tại.

Nếu chạy trên Windows thì chạy các câu lệnh sau trên terminal:
./exec.bat
./main *tên_file cho chương trình bai33.c
./exec_menu.bat cho chương trình bai33menu.c

Nếu chạy trên Linux thì chạy các câu lệnh sau trên terminal:
chmod 777 exec.sh
./exec.sh
./main *tên_file cho chương trình bai33.c

chmod 777 exec_menu.sh
./exec_menu.sh cho chương trình bai33menu.c
